# BitFex Token

## Requirements

Install truffle and ganache-cli:

```
npm install -g truffle
npm install -g ganache-cli
```

To install requirements run `yarn` in project directory.

## Tests

To run tests you need to run `testrpc`:

```
yarn testrpc
```

And run tests:

```
yarn test
```
